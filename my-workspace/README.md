# MyWorkspace Marvel Prueba Tecnica

App Marvel

## Paso 1

Run `npm install`

## Paso 2

Run `npm run app-view-watch` ejecuta la libreria.

## paso 3

Run `npm run start` ejecuta la app 

## Nota

Ninguna se debe cancelar , la libreria siempre debe estar arriba para que funcione la app
